
# Crawler - Node.JS
Este repósitorio tem o objetivo de apresentar este projeto.


## Como rodar na sua máquina 

Clone o projeto 

```bash
  git clone git@github.com:juliocesar014/crawler_node.git
```

Entre no diretório do projeto 
```bash
  cd crawler_node
```

Instale as dependências 

```bash
  npm install 
```

Inicie 

```bash
  node index.js
```
